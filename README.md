# 🚗 Smart Parking Detection System

An AI-powered Smart Parking Detection System built using **Python, OpenCV, Flask, HTML, CSS, and JavaScript**.

This project detects parking slot occupancy in real-time using computer vision techniques and displays live analytics on a modern web dashboard.

---

# ✨ Features

- Real-time parking slot detection
- Live parking video streaming
- Smart occupancy monitoring
- Interactive parking dashboard
- Available vs occupied slot analytics
- Dynamic occupancy percentage calculation
- Responsive frontend UI
- Flask API integration
- Parking slot selection tool

---
